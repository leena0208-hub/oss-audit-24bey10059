# The Open Source Audit: VLC Media Player
### OSS Capstone Project | VITyarthi

[cite_start]This repository contains a comprehensive audit and technical analysis of **VLC Media Player**, conducted as part of the Open Source Software (OSS) course[cite: 4, 12]. 

## 📌 Project Overview
The project is divided into two main components:
1. [cite_start]**The Audit Report**: A 12-16 page deep-dive into the philosophy, licensing (GPL/LGPL), and Linux footprint of VLC[cite: 24, 29].
2. [cite_start]**Shell Script Suite**: Five functional Bash scripts demonstrating Linux system administration, package inspection, and automation[cite: 89, 90].

## 📂 Repository Structure
* `/scripts`: Contains the five required shell scripts (.sh).
* `OSS_Audit_Report_VLC.pdf`: The complete analytical report.
* `README.md`: Project documentation and overview.

## 🛠️ Shell Scripts Included
1. [cite_start]**System Identity Report**: Displays Linux distribution, kernel version, and OSS license info[cite: 93].
2. [cite_start]**FOSS Package Inspector**: Verifies VLC installation and provides philosophical context using case statements[cite: 125].
3. [cite_start]**Disk & Permission Auditor**: Recursively checks system directories and VLC configuration paths[cite: 145].
4. [cite_start]**Log File Analyzer**: A tool to parse system logs for specific keywords like "ERROR"[cite: 163].
5. [cite_start]**Open Source Manifesto**: An interactive script that generates a personalized OSS philosophy statement[cite: 185].
